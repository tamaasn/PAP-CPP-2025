# Latihan Soal Fungsi Dasar C++

# 1. Fungsi Penjumlahan Dua Bilangan

Buat fungsi int tambah(int a, int b) yang mengembalikan hasil penjumlahan dua bilangan.

Contoh Input:

3
7

Contoh Output:

Hasil penjumlahan = 10

# 2. Fungsi Pengurangan Dua Bilangan

Buat fungsi int kurang(int a, int b) untuk menghitung selisih antara dua bilangan.

Contoh Input:

10
4

Contoh Output:

Hasil pengurangan = 6

# 3. Fungsi Perkalian Dua Bilangan

Buat fungsi int kali(int a, int b) yang mengembalikan hasil perkalian dua bilangan.

Contoh Input:

5
8

Contoh Output:

Hasil perkalian = 40


# 4. Fungsi Pembagian Dua Bilangan

Buat fungsi float bagi(int a, int b) yang mengembalikan hasil pembagian dua bilangan dalam bentuk
desimal.
Tambahkan pengecekan agar tidak terjadi pembagian dengan nol.

Contoh Input:

12
4

Contoh Output:

Hasil pembagian = 3

Contoh Input lainnya:

10
0

Output:

Error: pembagian dengan nol tidak diperbolehkan.

# 5. Fungsi Nilai Pangkat Bilangan

Buat fungsi int hasil_pangkat(int angka,int pangkat)yang mengembalikan nilai hasil pangkat dari
bilangan.

Contoh Input:

6
3

Contoh Output:

Hasil 6 pangkat 3 adalah 216


# 6. Fungsi Genap atau Ganjil

Buat fungsi bool isGenap(int x) yang mengembalikan true jika bilangan genap, dan false jika ganjil.

Contoh Input:

9

Contoh Output:

Bilangan 9 adalah ganjil

# 7. Fungsi Nilai Maksimum

Buat fungsi int maksimum(int a, int b) yang mengembalikan nilai terbesar dari dua bilangan.

Contoh Input:

12
20

Contoh Output:

Bilangan terbesar adalah 20

# 8. Fungsi Menghitung Volume Kubus

Buat fungsi int volumeKubus(int sisi) untuk menghitung volume kubus.
Rumus: V = sisi * sisi * sisi

Contoh Input:

3

Contoh Output:

Volume kubus = 27


# 9. Fungsi Konversi Suhu Celcius ke Fahrenheit

Buat fungsi float toFahrenheit(float c) untuk mengubah suhu dari Celcius ke Fahrenheit.
Rumus: F = (c * 9/5) + 32

Contoh Input:

25

Contoh Output:

Dalam Fahrenheit = 77

# 10. Fungsi Menghitung Luas Lingkaran

Buat fungsi float luasLingkaran(float r) untuk menghitung luas lingkaran.
Rumus: L = 3.14 * r * r

Contoh Input:

7

Contoh Output:

Luas lingkaran = 153.
